第一件事：看新台幣匯率（5分鐘）
打開匯率APP，看早上8:45報價。
只跟「前一天下午4點台灣外匯收盤價」比，不是美元指數。
使用鉅亨網搜「美元/台幣」→ 歷史資料 → 時間16:00。

標準：升值超過1角（0.1元）就算「明顯」。

明顯升值 → 外資錢進來，權值股（台積、0050）有機會
明顯貶值 → 外資匯出，今天別衝
平盤 → 正常，回到個股籌碼判斷

匯率就是外資的腳印。開盤前先看這個，你才不會被新聞帶風向。

1. 怎麼看「升貶的節奏」？（這個一定要做）
打開5分K線，盯著8:45那一根：
>如果緩步連3-5根小紅 → 外資分批匯入，通常會連買好幾天
>如果突然急拉一根大紅（5分鐘內升超過3分） → 可能是央行出手或單一鉅額，隔天很容易回貶，別追
>如果開盤直接跳空升（比前一天16:00收盤高5分以上） → 外資半夜就動了，今天開高機率高，但要小心開高走低  


2. 對照人民幣和韓元（這個也一定要做）
同時打開三個報價：台幣、人民幣、韓元。
如果只有台幣在升 → 可能是壽險或出口商拋匯，這種買盤不持續，別追高
如果三個一起升 → 國際資金真的在流入亞洲，今天大盤安全，甚至可以加碼
如果台幣貶、人民幣也貶，但韓元在升 → 小心！外資在賣台股、買韓股，台積電可能被倒貨

3. 抓央行防線
盯著32元整數關卡：
如果在32.000附近突然出現一堆買單掛著 → 央行在守，大盤容易盤整
如果32元被輕鬆突破，而且沒人守 → 央行暫時放手，外資至少連賣三天

4. 對照紐約盤收盤價
查前一晚凌晨3點紐約盤的美元/台幣收盤價。
跟紐約盤差不多 → 今天就是延續昨晚趨勢
比紐約盤強（台幣升更多） → 台灣本地有買盤，開盤通常會急拉一波
比紐約盤弱（台幣貶更多） → 台灣本地有人在拋，開盤可能直接殺，不要亂接刀  

第二件事：掃期貨夜盤
打開看盤軟體（XQ或三竹免費版就夠），看台指期夜盤收在哪裡、跟現貨差多少點。
抓100點：正價差超過100點 → 開高機率高；逆價差超過100點 → 開低機率高。

但有一個很重要的例外，我當初花了200點學費才搞懂：
每年6-8月除息旺季，台指期本來就會逆價差100-300點。那不是看空，是正常現象。要記得扣掉除息點數再判斷。我當初沒扣，看到逆價差就空下去，結果被軋了150點。
另外，重點不是看漲跌，是看有沒有「異常」。
舉例：如果美股大漲，但台指期夜盤沒怎麼動 → 台股相對弱勢，不建議追高。  

5. 看「價差變化的斜率」，不要只看數字（一定要做）
如果價差慢慢擴大（例如從+50爬到+120） → 市場有共識看多，開高之後通常會續強，可以追
如果最後一小時突然爆衝（例如從+80噴到+150） → 通常是程式單或軋空，隔天開高容易爆量出貨，不要追
如果高檔收斂（原本+150，收盤前縮到+100） → 有人在夜盤就先跑了，隔天容易開高走低

6. 觀察夜盤成交量（一定要做）
跟過去5天的平均成交量比：
爆量（超過1.5倍）而且價差擴大 → 大戶提前佈局，順著做勝率高
爆量但價差沒什麼動 → 多空在打架，開盤先等9:15方向出來再說
量縮 → 市場在等，開盤波動會很大，建議先看5分鐘再動作

7. 對照富台指
如果你有辦法看到富台指，可以比一下：
台指期正價差，但富台指逆價差 → 本土過度樂觀，外資反而保守，容易開高走低
台指期逆價差，但富台指正價差 → 本土過度悲觀，外資偷偷看多，可能開低走高，可以找買點  

8. 除息旺季的「扣點公式」（進階）
去證交所網站查「除息點數預估表」，路徑是：首頁→交易資訊→除息除權→預估影響點數。
舉例：如果今天預估除息點數180點，但台指期逆價差200點 → 扣掉180之後，實際上是正價差20點 → 偏多，不是偏空。


第三件事：掃關鍵分點
開盤前再確認一次。
打開籌碼K線（免費版就夠用），點「分點明細」，看過去5個交易日哪個分點連續買超或連續賣超。
免費版只能看近5天，而且資料會一直往更早滑掉。所以你週二才看，上週四的資料可能就不見了。我的解法：假日最後一天先把關鍵資料截圖存下來。不用花錢，花30秒就好。
重點盯那幾間有名的：
兆豐-嘉義（長線主力）
凱基-台北（短線高手）
永豐金-萬盛（波段王）
如果他們連續買超，而且買的量越來越大 → 放進觀察名單。
如果開始賣了 → 先跳過。
上週四大盤重挫，但外資反手大買友達18.23萬張。這就是「盤勢空但個股有籌碼」的經典案例。
啊不過我要說清楚：友達當天股價還是跌的。所以只是「值得觀察」，不要看到有人買就跟著衝。
我筆記本寫得很清楚：除非站回月線，否則不進場。  

9. 不是看「買超張數」，要看「買超金額」（一定要做）
把張數乘上股價：
如果買超張數很多，但都是低價股（10元以下） → 可能是主力在作價，風險高
如果買超張數不多，但金額很大（例如200張台積電，就是1.2億） → 這是真的大戶，值得追蹤

10. 用「連買天數」搭配「股價位置」（一定要做）
打開K線，看股價現在在哪：
如果在低檔（比月線低20%以上），而且分點連買3天 → 主力在摸底，可以小量跟，停損設5%
如果在高檔（比月線高20%以上），而且分點連買3天 → 小心這是最後出貨，反而要去看誰在賣
如果在盤整區（月線上下5%以內），而且分點連買5天 → 這是最甜的位置，主力在吸籌，等突破盤整區間就可以加碼

11. 觀察分點的「慣性」（進階）
點分點名稱，看過去三個月的紀錄：
隔日沖分點（像凱基-台北、元大-土城永寧） → 今天大買，明天九成會賣。你如果跟，隔天開高就要跑，不能抱
波段分點（像兆豐-嘉義、永豐金-萬盛） → 通常會連買3-5天，然後抱1-2個月，可以等拉回再進
避險分點（像港商麥格理、美銀） → 他們大買通常是為了掩護現貨賣超，不要跟  

12. 抓「假明牌」的破綻（高手區）
比對分點買超跟個股當天的走勢：
如果分點買超，但股價收黑K或上影線很長 → 代表有人趁分點買的時候在倒貨，隔天通常開低，不要碰

如果同一個分點同時出現在買超和賣超前五名 → 左手換右手製造假量，不要碰
這時候我才開始看新聞、看討論區。心裡有個底，知道今天風往哪吹。
